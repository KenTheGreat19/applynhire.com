# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-30

### Added
- Enterprise-level project structure
- TypeScript frontend with modular architecture
- FastAPI backend with router-based structure
- Comprehensive configuration files (.gitignore, .env.example, .editorconfig, .prettierrc, .eslintrc)
- CI/CD pipeline with GitHub Actions
- Testing infrastructure for both frontend and backend
- Service layer pattern for backend
- Path aliases for TypeScript imports
- API documentation with OpenAPI/Swagger
- Health check endpoint
- JWT authentication system
- Job CRUD operations
- User authentication (signup/signin)

### Changed
- Reorganized project into enterprise structure
- Updated tsconfig.json with better configuration
- Enhanced package.json with comprehensive scripts
- Improved backend organization with routers and services

### Documentation
- New comprehensive README.md
- Contributing guidelines (CONTRIBUTING.md)
- Code of Conduct (CODE_OF_CONDUCT.md)
- License file (LICENSE)
- API documentation in /docs

### Added
